# Project description

Tanhwe Guest House is a public accommodation and conference enquiry website with a protected operations application for a small hospitality team in Mukwe, Namibia. Guests browse live room inventory, check dates, and submit booking requests without creating accounts. Staff manage stays and assigned follow-ups; admins manage rooms, bookings, customers, documents, payments, and operational work; owners also manage access, settings, and financial reports.

The product should feel calm, warm, trustworthy, and practical. Prices, inclusions, availability expectations, phone, and WhatsApp actions must remain obvious on mobile.
