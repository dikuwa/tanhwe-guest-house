# Handover guide

Use the admin application for inventory, bookings, customers, documents, payments, and follow-ups. Owners alone manage roles, public settings, and reports. Deploy from a reviewed branch after applying migrations to a Neon branch and testing a Vercel preview. Keep a restore point before production migration. Store credentials only in platform environment settings. See `pre-deployment.md` for owner actions and `env-setup.md` for variables.
