# UI audit

## Public

The existing home page has strong imagery, clear contact actions, live featured rooms, availability search, breakfast, and conference content. V2 replaces the non-functional mobile menu with an accessible menu, updates heading typography, adds active navigation feedback, and expands room details into a multi-image gallery. Remaining production QA should verify real image crops and real inventory copy.

## Admin

The existing admin shell was compact and usable, but five routes were placeholders and financial visibility was incomplete. V2 uses consistent page headings, bordered tables, responsive overflow, actionable empty states, role-filtered navigation, mobile sign-out, owner-only reports/settings/users, customer profiles, documents, payments, and follow-up task cards. Very wide datasets still use deliberate horizontal scrolling.
