# Handover Guide

## Owner Training

Show the owner how to:

- Login
- View dashboard
- Add rooms
- Update room prices
- Upload room images
- View bookings
- Confirm/cancel bookings
- Generate quotes
- Generate receipts
- Record payments
- View financial summary
- Manage users
- Update settings

## Admin Training

Show admin how to:

- Manage bookings
- Add customers
- Follow up guests
- Send quotes/receipts
- Update room availability
- Block dates

## Staff Training

Show staff how to:

- View upcoming bookings
- Check in guests
- Check out guests
- Add notes
- View assigned follow-ups

## Maintenance Guide

Monthly:

- Check backups
- Review bookings
- Export customer records
- Review failed emails
- Check unused images
- Review activity logs

## Future Roadmap

Recommended future improvements:

- Online deposit payments
- SMS reminders
- Conference booking module
- Advanced reports
- Customer portal
- Booking.com/Airbnb sync
