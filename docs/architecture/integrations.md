# Integrations

## WhatsApp

Use direct links with prefilled messages.

Example message:

Hello Tanhwe Guest House, I would like to book a room from [check-in] to [check-out] for [guests] guests.

## Email

Use Resend for:

- Booking received
- Booking confirmation
- Quote sent
- Receipt sent
- Payment reminder
- Arrival reminder

## PDF Documents

Generate downloadable PDFs for:

- Quotes
- Receipts
- Invoices
