# Motion Rules

Use Framer Motion only where it improves usability.

Allowed:

- Page fade-in
- Card hover lift
- Dialog open/close
- Sheet slide
- Toast transitions
- Simple loading skeletons

Not allowed:

- Bouncing buttons
- Floating elements
- Scroll hijacking
- Heavy parallax
- Animated gradients
- Excessive reveal animations

Motion should be subtle and fast.

Recommended durations:

- Micro interactions: 120ms–180ms
- Modal/sheet: 180ms–240ms
- Page transitions: 200ms–300ms
